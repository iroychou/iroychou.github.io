Mac环境下配置Java环境:安装及配置JDK

发表于 2019-06-29

JDK下载

官网下载jdk,这里的版本是12.0.1,然后一步一步按提示安装。

安装成功之后按照下面的路径可以找到安装好的jdk:/Library/Java/JavaVirtualMachines/jdk-12.0.1.jdk

该jdk的根目录为Contents下的Home文件夹:

JDK环境变量配置

打开终端并输入vim ~/.bash_profile，回车进入编辑模式[~代表使用者的home目录，该隐藏的配置文件若不存在，vim指令会自动创建]。输入如下内容后退出编辑模式并保存:

JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk-12.0.1.jdk/Contents/Home
PATH=$JAVA_HOME/bin:$PATH:.
CLASSPATH=JAVA_HOME/lib/tools.jar:$JAVA_HOME/lib/dt.jar:.
export JAVA_HOME
export PATH
export CLASSPATH

至此，jdk配置完成。

  • 在终端输入java -version进行验证。
  • 输入echo $JAVA_HOME进行查看，结果则为刚才配置的jdk的根目录。
